## Résumé des changements
Installation de **PHP CS Fixer v3.95.2** comme outil de qualité de code en développement.

Le fichier `.php-cs-fixer.dist.php` est configuré avec :
- Le ruleset `@Symfony` (standard officiel Symfony)
- Exclusion du dossier `var/` et des fichiers générés (`config/bundles.php`, `config/reference.php`)

## Raison
Uniformiser le style du code PHP selon les conventions Symfony et prévenir les incohérences de formatage lors des contributions.

## Utilisation rapide
```bash
# Audit (sans modifier les fichiers)
vendor/bin/php-cs-fixer fix --dry-run --diff

# Correction automatique
vendor/bin/php-cs-fixer fix
```
